ШАГ 1: НАЧНИТЕ С ЭТОГО ФАЙЛА
→ Откройте START.md (2 минуты)

ШАГ 2: ЗАПУСТИТЕ УСТАНОВКУ
Windows:   .\setup.bat
Linux:     ./setup.sh

ШАГ 3: КОНФИГУРИРУЙТЕ
→ Отредактируйте файл .env
  - Добавьте BOT_TOKEN от @BotFather
  - Добавьте ваш Telegram ID в ADMIN_IDS

ШАГ 4: ЗАПУСТИТЕ БОТ
→ python main.py

ШАГ 5: ТЕСТИРУЙТЕ
→ Отправьте боту /start
→ Администратор выполнит /allow_user [ваш_ID]
