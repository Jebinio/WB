ИНСТРУКЦИЯ ПО ЗАПУСКУ:

1. ПОЛУЧИТЕ BOT_TOKEN
   ─────────────────────
   - Откройте Telegram
   - Найдите бота @BotFather
   - Отправьте команду: /newbot
   - Следуйте инструкциям
   - Скопируйте токен в формате: 1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefg

2. ПОЛУЧИТЕ СВОЙ TELEGRAM ID
   ─────────────────────────────
   - Найдите бота @userinfobot
   - Отправьте любое сообщение
   - Скопируйте ваш ID (число)

3. ОТРЕДАКТИРУЙТЕ ФАЙЛ .env
   ───────────────────────────
   - Откройте файл: c:\Users\Jeb\Desktop\WB\telegram_bot\.env
   - Замените YOUR_BOT_TOKEN_HERE на ваш токен
   - Замените YOUR_TELEGRAM_ID_HERE на ваш ID
   
   Пример:
   -------
   BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefg
   ADMIN_IDS=123456789

4. ЗАПУСТИТЕ БОТ
   ──────────────
   Откройте PowerShell в папке проекта и выполните:
   
   $env:PYTHONIOENCODING="utf-8"; python main.py

   ИЛИ запустите файл run_bot.bat (если создан)

5. ТЕСТИРУЙТЕ В TELEGRAM
   ──────────────────────
   - Добавьте бота в Telegram (найдите его по username из BotFather)
   - Отправьте команду /start
   - Администратор разрешит доступ командой: /allow_user [ваш_ID]
   - Готово! Все работает!
