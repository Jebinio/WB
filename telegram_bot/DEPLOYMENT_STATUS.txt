ЧТО ДАЛЬШЕ:
════════════════════════════════════════════════════════════════════════════════

ШАОГ 1: ПОЛУЧИТЕ BOT_TOKEN
─────────────────────────
Откройте Telegram и:
  1. Найдите @BotFather
  2. Отправьте /newbot
  3. Следуйте инструкциям
  4. Скопируйте токен

ШАОГ 2: ПОЛУЧИТЕ СВОЙ TELEGRAM ID
──────────────────────────────────
Откройте Telegram и:
  1. Найдите @userinfobot
  2. Отправьте любое сообщение
  3. Скопируйте ваш ID

ШАОГ 3: ОТРЕДАКТИРУЙТЕ ФАЙЛ .env
─────────────────────────────────
Откройте файл: c:\Users\Jeb\Desktop\WB\telegram_bot\.env

Замените:
  BOT_TOKEN=YOUR_BOT_TOKEN_HERE      на ваш токен
  ADMIN_IDS=YOUR_TELEGRAM_ID_HERE    на ваш ID

Пример:
  BOT_TOKEN=1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefg
  ADMIN_IDS=123456789

ШАОГ 4: ЗАПУСТИТЕ БОТ
──────────────────────
Способ 1 - Двойной клик:
  Откройте run_bot.bat в папке проекта

Способ 2 - PowerShell:
  $env:PYTHONIOENCODING="utf-8"; python main.py

ШАОГ 5: ТЕСТИРУЙТЕ
──────────────────
  1. Добавьте бота в Telegram (по username из BotFather)
  2. Отправьте /start
  3. Администратор выполнит: /allow_user ВАШ_ID
  4. Готово!
