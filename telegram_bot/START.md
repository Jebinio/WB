## 🚀 БЫСТРЫЙ СТАРТ

### 1️⃣ ДЛЯ WINDOWS
```powershell
# Откройте PowerShell в папке проекта и выполните:
.\setup.bat
```

### 2️⃣ ДЛЯ LINUX/macOS
```bash
# Откройте терминал в папке проекта и выполните:
./setup.sh
```

### 3️⃣ КОНФИГУРАЦИЯ
1. Откройте файл `.env`
2. Замените `YOUR_BOT_TOKEN_HERE` на токен от @BotFather
3. Замените `123456789` на ваш Telegram ID

### 4️⃣ ЗАПУСК
```bash
python main.py
```

### 5️⃣ ТЕСТ
- Отправьте боту `/start`
- Администратор выполнит `/allow_user [ВАШ_ID]`
- Всё работает! ✅
